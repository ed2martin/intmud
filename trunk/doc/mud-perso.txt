Personagens

* const tipoperso = 1
Indica o tipo de personagem:
1 = MOB, 2 = jogador

* txt1 sexo
Sexo, pode ser "O" ou "A"

* const nome = ""
Palavras que podem ser usadas para identificar o personagem.
Vide variável nomeobj do intmud.

* const descexec = ""
Descrição usada em ações com o personagem.

* const descpos = ""
Descrição quando o personagem estiver visível na posição padrão.

* const descver = ""
Descrição quando alguém olha para o personagem.

* const descr_<algumacoisa> = "texto"
Descrição de alguma coisa (descrição extra do personagem).

* listaitem dono
A sala onde o personagem está.

* listaobj dentro
Itens que está carregando ou usando.

* const dentromax = 10
Quantidade de itens que pode carregar.

* inttempo tempocair
Quanto tempo o personagem leva para cair ou afundar.

* uint16 p_nivel
Nível do personagem.

* uint16 p_exp
Pontos de experiência.

* uint32 p_moedas
Quantas moedas está carregando.

* uint32 p_banco
Quantas moedas possui no banco.

* int16 p_saude
* int16 p_mana
* int16 p_mover
Pontos de saúde (HP), mana e movimento.

* int16 p_saudemax
* int16 p_manamax
* int16 p_movermax
Pontos máximos de saúde, mana e movimento

* uint8 p_pos
Posição atual:
0 = morto
1 = mortalmente ferido
2 = incapacitado
3 = fraco
4 = dormindo
5 = descansando
6 = sentado
7 = lutando
8 = em pé

* const p_pospadr
Posição padrão; vide posicao, acima.

----------------------------------------
Não jogadores

* const p_mover = 1
Aonde pode se movimentar:
0 = não sai do lugar
1 = somente na área
2 = pode sair da área

* const p_pegar = 0
0 = não pega itens do chão
1 = pega itens
2 = pega somente itens de valor

* const p_agressivo = 0
0 = nunca ataca jogadores
1 = ataca se for atacado
2 = ataca sempre que possível

* const p_ajuda = 0
Ajuda outros mobs que estiverem sendo atacados por jogadores.

* const p_foge = 0
Se for agressivo: foge quando estiver fraco.
Se não for agressivo: foge quando for atacado.

* const p_naodorme = 0
Se não pode ser colocado para dormir

* const p_costas = 0
Se pode ser atacado pelas costas

* const p_seguir = 0
Se pode ser seguido.

* const p_nadar = 0
Pode atravessar lugares em que não se pode nadar.

* const p_silencio = 0
Se movimenta em silêncio (não é informado na sala).

* const p_imortal = 0
Se não pode morrer.

* const p_compra = 0
Porcentagem de lucro quando compra: preço final = valor * (100-v_compra)

* const p_venda = 0
Porcentagem de lucro quando vende: preço final = valor * (100+v_compra)

* const p_compravenda = 0
Se está aberto para compra e venda.

----------------------------------------
Jogadores

* txt20 p_senha
Senha

* uint8 p_conect
Estado da conexão:
0 = jogando
1 = usuário desconectado
2 = esperando apelido
3 = esperando senha

* txt60 p_sala
Nome da sala onde está.
