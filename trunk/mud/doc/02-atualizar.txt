1. Atualizar o MUD a partir de versões anteriores ao IntMUD 1.03

É sempre recomendável manter uma cópia da versão anterior por um tempo.

Apague os arquivos do diretório config da nova versão.

Copie as áreas que você criou (arquivos mud-a-*.int) para o diretório
config da nova versão, com exceção dos seguintes arquivos:
mud-a-casa.int
mud-a-fim.int
mud-a-ini.int
mud-a-teste.int

Copie todos os arquivos mud-b-*.int da versão antiga para o diretório
config da nova versão. São eles:
mud-b-config.int
mud-b-tipo.int
mud-b-social.int
mud-b-hab.int
mud-b-efeito.int

Copie os diretórios sav e sav2 (os jogos salvos) da versão antiga para a nova.

Na nova versão, renomeie o diretório sav para sav1.

Rode o MUD. Se for necessário, ele atualizará os arquivos para a nova versão.


2. Atualizar o MUD a partir de versões do IntMUD 1.03 em diante

É sempre recomendável manter uma cópia da versão anterior por um tempo.

Copie os arquivos do diretório areas da antiga versão para a nova,
com exceção dos seguintes arquivos:
casa.int
fim.int
ini.int
teste.int

Apague os diretórios config, sav1 e sav2 da nova versão.

Copie os diretórios config, sav1 e sav2 da antiga versão para a nova.
