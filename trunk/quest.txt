*** Introdução

Quest é um jogo de questionários. São perguntas com alternativas,
sendo apenas uma correta. O objetivo é promover o aprendizado, não
avaliar os conhecimentos de cada um.

O usuário escolhe o questionário (o jogo suporta até 10).
As perguntas são apresentadas em uma ordem aleatória.
Quando o jogador responde corretamente, a pergunta é retirada da lista.
Se responder errado, o jogo mostra qual era a alternativa certa e
acrescenta a pergunta novamente na lista.

Antes de cada pergunta é exibida a quantidade de perguntas que faltam
(um número). Quando chega a zero, a pontuação é calcula, de acordo com
quantidade de erros e no tempo que o usuário levou para terminar
o questionário.

*** Lista de arquivos

quest.int - o jogo em si, feito em IntMUD
intmud.exe - necessário para rodar o arquivo quest.int
quest0.txt a quest9.txt - perguntas e respostas dos níveis 0 a 9
quest.sav - contém os récordes; é criado automaticamente pelo jogo

*** Como rodar o jogo

Há três formas de rodar o jogo:
1. Independente. No próprio micro do usuário, sem a necessidade de
programas externos.
2. Como servidor de Telnet. Os jogadores usam um programa de Telnet,
ou o próprio Papovox.
3. Como bot do Minichat ou do Bate-papo SACI na Intervox.
O jogo é ativado dentro do chat.

A forma como o programa vai rodar é definido no início do arquivo
quest.int. Inicialmente vem configurado para rodar independente.

Os passos para rodar o jogo são:

1. Se necessário, abrir o arquivo quest.int em um editor de texto
e alterar a partir da linha 5 do arquivo. As seguintes linhas:
const modo = 0
const servporta = 1963
const serv = "localhost"
const porta = 1963
const sala = "sala01"
const senha = "sala01"
O jogo vem configurado para executar localmente, em uma janela.

2. Executar:
intmud quest.int

Se estiver usando algum ambiente gráfico, basta arrastar o arquivo
quest.int para o intmud.exe com o botão esquerdo do mouse.

*** Arquivos de perguntas e respostas

São os arquivos quest0.txt a quest9.txt (níveis 0 a 9).
A primeira linha contém sempre o título, que aparece ao escolher
a opção I (informações) durante o jogo.
Da segunda em diante são as perguntas e respostas.
Linhas que começam com a letra P e um espaço contém perguntas.
Linhas que começam com a letra R e um espaço contém respostas.
Após cada pergunta deve-se ter de uma a dez respostas.
Somente uma resposta: o usuário deve digitar a resposta.
Mais de uma resposta: é uma pergunta de alternativa, sendo a
primeira correta e as demais incorretas. O jogo encarrega-se
de exibir as respostas em uma ordem aleatória.
Quando o texto (pergunta ou resposta) não couber em uma linha,
pode-se continuar na próxima, começando a próxima linha com
a letra C e um espaço.
Qualquer outra linha que não começa com as letras P, R ou C,
seguidas de um espaço, é ignorada.
